Each branch is an exploration axis. Order adapts to the domain and block type. All applicable branches must be covered unless the user opted for a shorter session.

All questions, options, and signals must use the language chosen in Step 1. Examples below are in English — translate at execution time.

## Branch 1: The Block (always required)

What exactly is stuck? Surface the real obstacle.

**Sample questions:**
- "What's the block you're experiencing right now?"
  - Options derived from the initial description (e.g., "Can't choose between two paths", "Lost motivation", "Overwhelmed by options", "Fear of making the wrong choice")
- "When did this block start?"
  - Options: "Recently — triggered by a specific event", "It's been building for a while", "It's always been there", "Not sure"
- "What have you already tried to get unstuck?"
  - Options derived from context + "Nothing yet" + "Several things, none worked"

**Sub-branches:**
- If triggered by event → explore what changed
- If chronic → explore what keeps it in place
- If "nothing tried" → explore what's preventing action

**Minimum questions: 2–3**

## Branch 2: Desired Outcome (always required)

Where do you want to be? What does "unblocked" look like?

**Sample questions:**
- "If this block disappeared tomorrow, what would be different?"
  - Options derived from context (e.g., "I'd start the project", "I'd have a clear direction", "I'd feel lighter", "I'd make the decision")
- "What does success look like for you here?"
  - Options: concrete outcomes derived from the domain (e.g., for career: "New role by Q3", "Clear growth plan", "Peace with current situation")
- "How will you know you've made progress?"
  - Options: measurable signals derived from context + "I'll feel it" + "Not sure yet"

**Sub-branches:**
- If vague outcome → explore what "better" means concretely
- If multiple outcomes → explore which one matters most

**Minimum questions: 2–3**

## Branch 3: Root Causes (always required)

What's underneath the surface? Beliefs, fears, patterns.

**Sample questions:**
- "What's the worst thing that could happen if you [action related to block]?"
  - Options derived from context (e.g., "I'd fail publicly", "I'd waste time", "I'd disappoint someone", "Nothing terrible, actually")
- "Is there a belief holding you back?"
  - Options: common beliefs for the domain (e.g., for creative: "I'm not talented enough", "It's been done before", "Who am I to do this?", "I don't see one")
- "Have you been in a similar block before? What happened?"
  - Options: "Yes, and I got through it by...", "Yes, and I'm still dealing with it", "No, this is new", "I see a pattern"

**Sub-branches:**
- If fear identified → explore its validity (is it real or assumed?)
- If pattern identified → explore what's different this time
- If "nothing terrible" → explore what's really keeping them stuck

**Minimum questions: 2–4**

## Branch 4: Context & Constraints (always required)

What can't change? What resources exist?

**Sample questions:**
- "What constraints are non-negotiable?"
  - Options derived from domain (e.g., for career: "Financial obligations", "Location", "Family needs", "Timeline") + "None, I'm flexible"
- "What resources do you have that you might not be using?"
  - Options: "People who could help", "Skills I'm undervaluing", "Time I could reallocate", "Money/savings", "I'm not sure"
- "What would make this easier?"
  - Options derived from context + "Permission from myself" + "More information" + "A clear first step"

**Sub-branches:**
- If strong constraints → explore creative ways within them
- If "permission from myself" → this is often the real block — explore

**Minimum questions: 2–3**

## Branch 5: Perspectives (optional — include for complex blocks)

How do others see this? What angles are you missing?

**Sample questions:**
- "If someone you admire faced this same block, what would they do?"
  - Options: specific role models if mentioned + "They'd just start", "They'd ask for help", "They'd reframe the problem", "I don't know"
- "What would your best friend say about this situation?"
  - Options derived from context (e.g., "You're overthinking it", "You already know the answer", "Try X first", "Take a break")
- "Is there a perspective you've been avoiding?"
  - Options derived from contradictions found in earlier answers + "Not that I can think of"

**If documents/research were provided:**
- "The [source] suggests [perspective]. How does that land?"
  - Options: "That resonates", "Interesting but doesn't apply", "Actually, that challenges my assumption", "I want to explore that more"

**Minimum questions: 2–3**

## Branch 6: Connections & Patterns (optional — include when documents/research provided)

Cross-reference all inputs to find non-obvious links.

**Sample questions:**
- "I notice a pattern between [X from your answers] and [Y from the document/research]. Does that connection feel meaningful?"
  - Options: "Yes, I hadn't seen that", "Interesting but coincidental", "That's exactly the issue", "Tell me more"
- "The research shows that people in similar situations often [finding]. Does that match your experience?"
  - Options: "Exactly", "Partially", "Not at all — my situation is different because...", "I want to explore why"

**Minimum questions: 1–2 (depends on material available)**

## Branch 7: Clarity & Action (always required)

What's the insight? What's the first step?

**Sample questions:**
- "Based on everything we've explored, what feels clearest to you now?"
  - Options: key insights surfaced during the session + "I need to sit with this" + "I see it differently now"
- "What's the smallest possible next step you could take?"
  - Options: concrete micro-actions derived from the session (e.g., "Write down my decision", "Have one conversation", "Spend 15 minutes on [X]", "Let go of [Y]")
- "What do you want to remember from this session?"
  - Options: top insights as summary phrases + "All of it" + "I'll know when I read the output"

**Minimum questions: 2–3**

## Suggested Order by Domain

**Career / Professional:** Block → Desired Outcome → Constraints → Root Causes → Perspectives → Action
**Creative:** Block → Root Causes → Desired Outcome → Perspectives → Constraints → Action
**Business / Strategy:** Block → Constraints → Desired Outcome → Perspectives → Root Causes → Action
**Relationships / Social:** Block → Root Causes → Perspectives → Desired Outcome → Constraints → Action
**Health / Wellness:** Block → Desired Outcome → Constraints → Root Causes → Perspectives → Action
**Learning / Development:** Block → Desired Outcome → Root Causes → Constraints → Perspectives → Action

Order may change if the conversation naturally leads elsewhere. The important thing is covering all required branches.
